# Javascript Sayı Tahmin Oyunu
JavaScript ile oluşturulmuş basit bir sayı tahmin oyunu

## Nasıl Kurulur
Projeyi makinenize klonlamanız yeterlidir.

## Nasıl oynanır
- index.html dosyasını favori tarayıcınızda açın.
- Kutuya seçtiğiniz bir numarayı girin.
- Tahmin Et düğmesine tıklayın.
- Seçtiğiniz sayı rastgele sayı ile eşleşirse, kazanırsınız

[Dene](https://nnumberguessinggame.netlify.app/)
